## 강원도에 위치한 생산공장 목록 출력하기
```sql
SELECT FACTORY_ID, FACTORY_NAME, ADDRESS
FROM FOOD_FACTORY
WHERE ADDRESS LIKE "%강원도%"
ORDER BY FACTORY_ID
```
### 개념학습
#### 특정 문자 포함여부로 필터링
- 기본문법 : `SELECT * FROM [테이블명] WHERE LIKE [조건]`
- LIKE 조건: -, %, [] 사용
  1. `LIKE "_A"` -> A로 끝나는 두 자리 문자 유무("_"는 자리수 지정)
  2. `LIKE "%A"` -> A로 끝나는 문자 유무 ("%"는 자리수 지정 X)
  3. `LIKE "[^A]%"` -> [^A]; "A"가 아닌 문자 집합, A로 시작하지 않는 문자 유무
 
### 기억하기
특정 문자의 포함여부는 **WHERE LIKE [조건]** 사용한다.
