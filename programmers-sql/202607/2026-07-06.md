## 1. 강원도에 위치한 생산공장 목록 출력하기
```sql
SELECT FACTORY_ID, FACTORY_NAME, ADDRESS
FROM FOOD_FACTORY
WHERE ADDRESS LIKE "%강원도%"
ORDER BY FACTORY_ID
```
### 개념학습
a. 특정 문자 포함여부로 필터링
- 교과서 문법: `SELECT * FROM [테이블명] WHERE LIKE [조건]`
- 조건식 작성: -, %, [] 사용
  1. `LIKE "_A"` -> A로 끝나는 두 자리 문자 유무("_"는 자리수 지정)
  2. `LIKE "%A"` -> A로 끝나는 문자 유무 ("%"는 자리수 지정 X)
  3. `LIKE "[^A]%"` -> [^A]; "A"가 아닌 문자 집합, A로 시작하지 않는 문자 유무 (SQL Server)
     <-> (MYSQL) `WHERE 컬럼명 REGEXP '^[^A]'`
   
### 기억하기
특정 문자의 포함여부는 **WHERE LIKE [조건]** 사용한다.

## 2. 경기도에 위치한 식품창고 목록 출력하기
```sql
SELECT WAREHOUSE_ID, WAREHOUSE_NAME, ADDRESS, COALESCE(FREEZER_YN,'N')
FROM FOOD_WAREHOUSE
WHERE ADDRESS LIKE "%경기도%"
ORDER BY WAREHOUSE_ID
```
### 개념학습
a. COALESCE()
- 교과서 문법: `COALESCE(인자1, 인자2, .. 인자N)` -> 인자 목록을 왼쪽부터 확인하여 NULL이 아닌 첫 번째 값을 반환함.
- 코드 문법 : `COALESCE(col, 대체값)` -> NULL이 아니면 col값을, NULL이면 대체값을 반환함.
- `COALESCE(FREEZER_YN, 'N') AS FREEZER_YN` 로 컬럼명도 원래처럼 지정 가능

b. COALESCE() vs CASE WHEN
- COALESCE() -> 컬럼의 NULL값을 특정 기본값으로 대체할 떄 사용
- CASE WHEN ->  컬럼의 값을 조건별로 매핑•대체할 떄 사용
  ```
  SELECT
      CASE
          WHEN 조건1 THEN 값1
          WHEN 조건2 THEN 값2
          ELSE 기본값
      END AS col_name;
  ```
### 기억하기
결측값을 하나의 값으로 대체할 때는 **SELECT COALESCE()** 사용한다.

## 3. 조건에 맞는 회원수 구하기
```sql
SELECT COUNT(*) AS CNT
FROM USER_INFO
WHERE JOINED BETWEEN '2021-01-01' AND '2021-12-31' AND AGE BETWEEN 20 AND 29
```

### 개념학습
a. COUNT()
- 교과서 문법
  - COUNT(*) -> 테이블의 행 개수
  - COUNT(col_name) -> 컬럼에서 NULL 값 제외한 값의 개수
  - COUNT(DISTINCT col_name) -> 컬럼에서 NULL 값, 중복 제외한 값의 개수
b. 비교연산
- `col_name BETWEEN 조건1 AND 조건2` 사용
- `WHERE 조건1<=col_name<=조건2`로 direct 비교 불가능함.

### 기억하자
출력되는 결과의 개수를 집계할 때는 **SELECT COUNT()** 사용한다.

## 4. 가장 비싼 상품 구하기
```sql
SELECT MAX(PRICE) AS MAX_PRICE
FROM PRODUCT;
```

### 개념학습
a. 최댓값이 필요한 경우 -> MAX(col) 사용
b. 최댓값이 존재하는 행이 필요한 경우 -> TOP N, LIMIT 사용
  - SQL Server
  ```
  SELECT TOP 1 *
  FROM PRODUCT
  ORDER BY PRICE DESC;
  ```
  - MySQL
  ```
  SELECT *
  FROM PRODUCT
  ORDER BY PRICE DESC
  LIMIT 1;
  ```
### 기억하자
최댓값이 필요한지 최댓값이 존재하는 행이 필요한지 구분하자.
