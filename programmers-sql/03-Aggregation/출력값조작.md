## 문자열 병합 출력: CONCAT()
---
- **기본문법:**
```sql
CONCAT(문자열1, 문자열2, ..)
-- 여러 문자열 (문자열 컬럼s OR 문자열 컬럼 + 문자열)을 하나의 문자열로 병합하여 출력
```

- **사용예제:**
```sql
SELECT CONCAT(col, 'cm')
SELECT CONCAT(col1, '::', col2) AS new_col
```

## 반올림/올림/내림/버림 출력
---
- **어림셈 종류**: ROUND(), CEIL(), FLOOR(), TRUNC()
- **기본문법**
```sql
ROUND(num_col, loc=0)         -- loc는 소숫점 자릿수, 기본은 정수출력
CEIL(num_col), FLOOR(num_col) -- 가까운 정수로 올림/내림
TRUNC(num_col, loc=0)         -- loc까지 남기고, 나머지 숫자는 버림
```

- **사용예제:**
```sql
SELECT CEIL(34.567), CEIL(-101.1), CEIL(101.01)           -- 35, -102, 102
SELECT TRUNC(34.567), TRUNC(-34.567,1), TRUNC(-34.567,-1) -- 34, -34.5, -30⭐

```
