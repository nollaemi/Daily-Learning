## 결측치를 단일 값으로 대체: COALESCE()

---
- **기본문법:**
```sql
COALESCE(인자1, 인자2, .. 인자N)
-- 인자 목록을 왼쪽부터 확인하여 NULL이 아닌 첫 번째 값을 반환
```
- **사용예시:**
```sql
SELECT COALESCE(col, val) AS new_col
-- 컬럼의 값이 NULL이 아니면 원본값, NULL이면 val값을 반환
```

## 조건에 따라 새로운 컬럼 값 매핑: CASE WHEN THEN ELSE END
- 결측값을 복잡한 조건으로 대체할 때 사용 가능함.
- **기본문법:**
```sql
SELECT
  CASE
    WHEN cond1 THEN val1, -- 조건은 old_col에 관한 것
    WHEN cond2 THEN val2,
    ELSE val3
  END AS new_col
```
